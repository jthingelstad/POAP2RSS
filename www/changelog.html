Changelog

  • 2025-07-04 – Initial public release of POAP2RSS. Celebrated with a POAP of course.
  • 2025-07-05 – Revamped website.
  • 2025-07-11 – Moved RSS limit from 20 to 50 items. Changed inactive notifications to start at 12 weeks instead of 4 weeks.
  • 2026-01-11 – Added ?nowarning=1 parameter to suppress inactivity alerts. Added URL converter tool on homepage. Added pagination for event feeds supporting more than 100 claims. Added MAX_CLAIMS_COUNT environment variable. Added status page link and deployment script.
  • 2026-01-12 – Added Tinylytics analytics to the website.
  • 2026-02-15 – Added Tinylytics hit tracking to the Lambda function. Redesigned website with CRT green screen theme. Added analytics link to navigation.

This project is open source. Pull requests are accepted.

You can also see blog posts about POAP2RSS for more information.
